# Personal Finance Power BI Report

## Summary
Fully interactive Personal Finance Report pulling from an Excel file. Data shown in Live Demo has been randomized or flattened. Users can download Excel file and Power BI Template file to enter their own data. Report contains pages broken out by the account level, credit statement information, retirement, and more.

Keep reading below for a full description of each page, user guide for the Excel data template, and a list of all calculated column and measures.
